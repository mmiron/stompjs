# Onboarding Checklist — Landscaping Members (v1)

Purpose: Standard onboarding flow for landscaping/lawn-care members  
Target onboarding window: 7–14 days

---

## 1) Pre-Kickoff (Day 0)
- [ ] Signed membership agreement received
- [ ] Billing setup completed
- [ ] Primary owner contact confirmed
- [ ] Support channel confirmed (email/portal)

## 2) Kickoff Call (Day 1)
- [ ] Confirm business model (recurring vs one-time jobs)
- [ ] Confirm seasonality profile and peak months
- [ ] Confirm top 3 admin pain points
- [ ] Define 30-day success outcomes
- [ ] Confirm scope (in-scope/out-of-scope)

## 3) Data and Access Collection (Days 1–3)
- [ ] Current customer list imported
- [ ] Existing recurring billing schedule captured
- [ ] Current invoice templates collected
- [ ] Accounting tool access granted (or exports provided)
- [ ] Website/hosting credentials documented (if in-scope)
- [ ] Legal/accounting intake baseline completed

## 4) Workflow Setup (Days 3–7)
- [ ] Recurring invoice cadence configured
- [ ] Payment reminder cadence configured
- [ ] Monthly reconciliation checklist configured
- [ ] Missed/paused service handling process configured
- [ ] Weather/season exception rules documented
- [ ] Request routing and SLA lane activated

## 5) Partner Coordination (Days 5–10)
- [ ] Accounting partner lane activated (if needed)
- [ ] Legal intake pathway configured (if needed)
- [ ] Escalation contacts documented
- [ ] SLA targets confirmed with member

## 6) Quick-Win Validation (Days 10–14)
- [ ] First recurring billing run completed
- [ ] Aging invoices and follow-ups reviewed
- [ ] Member receives first “ops health” snapshot
- [ ] First optimization recommendations delivered

## 7) 30-Day Success Review
- [ ] Admin time saved estimate captured
- [ ] Billing consistency improvement captured
- [ ] Open issues list reviewed and closed/assigned
- [ ] Next-30-day action plan agreed

---

## Segment-Specific Risks to Watch
- Seasonal volatility can disrupt recurring billing assumptions
- High field-work load can delay owner responses
- Service pauses/cancellations must be handled quickly to avoid billing errors

## Segment-Specific KPI Focus
- On-time recurring invoice rate
- Days-to-cash trend
- Support requests related to billing exceptions
- 30-day member satisfaction signal
