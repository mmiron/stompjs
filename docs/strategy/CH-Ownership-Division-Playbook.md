# Ownership Division Playbook (Cofounders)

Date: March 3, 2026  
Scope: Practical equity split framework for Collective Hive cofounder scenarios

---

## 1) Core Principle

Ownership should reflect **future value creation**, not only who had the original idea.

Use four weighted factors:
- Commitment (time and duration)
- Role criticality (impact on success)
- Execution capability (ability to deliver)
- Risk/capital contribution (cash, deferred salary, reputational/financial risk)

---

## 2) Recommended Structural Defaults

- Reserve an **employee option pool** early (typically 10–20%)
- Put all founder equity on **vesting** (standard: 4 years, 1-year cliff)
- Use **good leaver / bad leaver** terms
- Keep major decisions governed by reserved-matter rules

---

## 3) Suggested Scoring Weights

- Commitment: 35%
- Role criticality: 30%
- Execution track record: 20%
- Risk/capital contribution: 15%

Scoring process:
1. Score each founder 1–10 in each category
2. Apply weights
3. Convert weighted scores into percentage split within founder pool
4. Pressure-test for fairness and motivation

---

## 4) Scenario Options (Illustrative)

## Scenario A — You + 1 Full-Time Ops Cofounder
Use when there are two true full-time cofounders and you remain primary founder/CEO.

Example ranges:
- Founder (you): 55–65%
- Ops cofounder: 35–45%

Example target split:
- You: 60%
- Ops: 40%

## Scenario B — You + Ops + Product/Automation Cofounder (All Full-Time)
Use when three cofounders are all core to execution.

Example ranges:
- You: 45–55%
- Ops: 25–35%
- Product/Automation: 15–30%

Example target split:
- You: 50%
- Ops: 30%
- Product/Automation: 20%

## Scenario C — You + 1 Full-Time Cofounder + 1 Part-Time Strategic Cofounder
Use when one partner is materially lower commitment.

Example ranges:
- You: 55–70%
- Full-time cofounder: 25–40%
- Part-time strategic cofounder: 5–15%

Example target split:
- You: 62%
- Full-time partner: 30%
- Part-time partner: 8%

---

## 5) Option Pool Planning

Two ways to structure:

### Method 1 (Cleaner for planning)
- First define founder split = 100%
- Then dilute all founders equally to create option pool (e.g., 15%)

### Method 2 (Negotiation-first)
- Include option pool in cap table from day one
- Negotiate founder percentages post-pool

Important: be explicit which method you’re using to avoid confusion.

---

## 6) Vesting and Protection Terms (Must-Have)

- Vesting: 48 months
- Cliff: 12 months
- Monthly vesting after cliff
- Unvested shares repurchased on departure
- IP assignment required from all founders
- Confidentiality and non-solicit clauses

---

## 7) Decision Rights Framework

Role-owner decisions:
- Routine functional decisions in approved budget/scope

Reserved matters (majority/unanimous):
- Equity issuance
- Debt above threshold
- Major hiring/firing
- Material strategic pivot
- Company sale or asset transfer

---

## 8) Fairness Stress Test

Before finalizing split, ask:
- Would each founder still accept this split after 18 months of hard execution?
- Does split match real workload and accountability?
- Does split protect business continuity if one founder exits?
- Is there enough option pool to hire critical talent?

If any answer is “no,” iterate before signing.

---

## 9) Practical Recommendation for Your Current Context

Given your role as initiator/vision/GTM and expected need for ops-heavy execution:

- If 2 founders (you + ops): start discussions around **60/40**
- If 3 founders (you + ops + product): start discussions around **50/30/20**
- If third partner is part-time: keep them in **5–12%** range with strict vesting/milestones

These are starting points, not legal prescriptions.
